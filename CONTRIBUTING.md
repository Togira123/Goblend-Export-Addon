# Contributing

Glad to hear that you want to contribute!

Just a few things that are important:

### General

The project uses the [python black formatter](https://github.com/psf/black).
Make sure to format your code with it and make sure to include the config file for it (pyproject.toml)
by running the formatter with the `--config pyproject.toml` command line argument.

If you're implementing a new feature, create an issue for it first where it can be discussed.
It is a waste of your time if you implement a feature that is not wanted in the first place.

### Coding Agent / AI Use

Goblend is strictly *not* vibe-coded, meaning no coding agents or other forms of AI were used to generate code.
I'd like it to stay this way, hence no pull requests will be accepted that use clearly AI generated code.
Using AI as a research tool is fine, but please write the final code yourself.
